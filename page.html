Présentation

Windows est une interface graphique produite par Microsoft, qui est devenue ensuite un système d'exploitation, principalement destinés aux ordinateurs compatibles PC. La gamme Windows est composée de plusieurs branches : La première branche, dite branche 16 bits, couvre Windows 1 à 3.11. Elle est apparue en 1985 et fonctionnait uniquement sur compatibles PC, en mode 16 bits ; La deuxième branche, dite branche Windows NT, est apparue en 1993. C’est un développement repartant de zéro, destiné aux ordinateurs personnels, aux serveurs et à des ordinateurs non compatibles PC. Elle a d’abord été utilisée dans les entreprises. Avec Windows XP, sorti en 2001, qui continue la branche Windows NT, cette branche est désormais aussi grand public, et se poursuit avec Windows Vista, Windows 7 et Windows 8 et Windows 10 ; La troisième branche, parfois appelée branche Windows 9x, est apparue en 1995 et a existé avec la branche NT. Cette branche a débuté avec Windows 95, suivi de Windows 98 et Windows Me. C’est la première branche grand public compatible 32 bits; La quatrième branche, dite branche Windows CE, apparue en 1996 avec Windows CE 1.0. Elle est destinée aux systèmes embarqués et matériels légers et portables (assistant personnel, téléphone portable). C’est la base de Windows Mobile et Pocket PC ; La cinquième branche, Windows RT, conçue exclusivement pour les processeurs ARM.
